Bootstrap Layout

Modelo de contenedor y modelo de maya de Bootstrap

Bootstrap utiliza un modelo de contenedor y modelo de maya para establecer un sistema de grilla que permite organizar el contenido de la página en filas y columnas. El modelo de contenedor es un contenedor que envuelve todo el contenido de la página y define el ancho máximo de la página. El modelo de maya se basa en un sistema de 12 columnas, donde cada columna tiene un ancho relativo en porcentaje.
